# ReBase

A modern, full-stack task management application built with React, TypeScript, and Supabase. This application provides real-time task synchronization, image uploads, and a beautiful, responsive user interface.

## Features

- **Authentication**: Secure user authentication with email/password using Supabase Auth
- **Task Management**: Create, read, update, and delete tasks
- **Image Upload**: Attach images to tasks with Supabase Storage
- **Real-time Updates**: Live task synchronization across multiple clients using Supabase Realtime
- **Modern UI**: Beautiful, responsive design with Tailwind CSS and Framer Motion animations
- **Toast Notifications**: User-friendly feedback with Sonner toast notifications
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
